# 🌍 Ready Safe – Disaster Preparedness Information Hub

## 📌 Overview
**Ready Safe** is a simple and mobile-friendly disaster preparedness information hub designed to provide fast, clear, and accessible emergency guidance.

This project helps users quickly understand what to do **before, during, and after disasters**, improving awareness and readiness during critical situations.

---

## 🚨 Problem Statement
Many people struggle to find clear and immediate instructions during emergencies. Information is often:
- Hard to access quickly
- Too complex or technical
- Not mobile-friendly

This can lead to confusion and delayed responses during disasters.

---

## 💡 Solution
Ready Safe solves this by offering:
- Simple and clean interface
- Easy navigation for different disaster types
- Quick access to safety procedures
- Mobile-friendly responsive design

---

## 🖥️ Features
- 🏠 Home hero landing page
- 🌪️ Disaster categories:
  - Earthquake
  - Typhoon
  - Flood
  - Fire
  - Emergency contacts
- 📱 Responsive design for mobile and desktop
- 🎨 Clean UI with visual elements and illustrations
- ⚡ Fast access to safety instructions

---

## 🧭 Sample Hero Page
The landing page introduces the project with a strong message:

> **“Be Ready, Be Safe”**

It visually communicates preparedness and encourages users to take action before disasters happen.
